Hello MGL7361
=============

Application simpliste illustrant l'utilisation de [Ant][ant] et de [Ivy][ivy].

Prérequis
---------

Java 6 et Ant doivent être installés.

Marche à suivre
---------------

    $ git clone 
    $ cd hello-mgl7361
    $ ant

[ant]: http://ant.apache.org/
[ivy]: http://ant.apache.org/ivy/
